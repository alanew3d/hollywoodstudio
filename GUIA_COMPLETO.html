2
Vercel — variáveis de ambiente (as obrigatórias)
AGORA · 15 min
🔧
Onde configurar no Vercel
Vercel Dashboard → seu projeto → Settings → Environment Variables → Add
▸ Abrir Vercel Dashboard
📋
Variáveis obrigatórias — copie e preencha
Adicione uma por vez. Nome exato à esquerda, valor à direita.
ADMIN_SECRET
Valor: uma senha forte que você escolhe — anote em local seguro. Esta é a senha para salvar configs pelo admin.
ADMIN_EMAIL
Valor: seu email (alan@ideiaprojects.com) — marca como admin no login Google.
SITE_URL
Valor: https://hollywoodstudio.vercel.app (ou seu domínio quando conectado).
STRIPE_SECRET_KEY
Valor: começa com sk_live_... — vem do painel Stripe.
STRIPE_WEBHOOK_SECRET
Valor: começa com whsec_... — gerado ao criar o webhook no Stripe.
GOOGLE_CLIENT_ID
Valor: termina em .apps.googleusercontent.com — vem do Google Cloud Console.
🗄️
Ativar Vercel KV (banco de dados grátis)
É necessário para salvar configurações e créditos entre sessões.
▸ Vercel Dashboard → seu projeto → Storage → Create → KV Store → Create
Depois de criar: clique em "Connect to Project" no KV criado. As variáveis KV_REST_API_URL e KV_REST_API_TOKEN são adicionadas automaticamente. Não precisa copiar nada.
3
Google OAuth — login com Google
AGORA · 20 min
🔑
Criar Client ID no Google Cloud Console
▸ Abrir Google Cloud Console → Credenciais
Passo a passo:
1. Criar credenciais → ID do cliente OAuth
2. Tipo: Aplicativo da Web
3. Nome: Hollywood Studio AI
4. Origens JavaScript autorizadas — adicionar as duas:
https://hollywoodstudio.vercel.app
https://hollywoodstudio.ai
5. Salvar → copiar o Client ID (formato: números-letras.apps.googleusercontent.com)
6. Colar como variável GOOGLE_CLIENT_ID no Vercel (Fase 2) e no admin do site
Tela de consentimento OAuth: Antes de criar o Client ID, configure em "Tela de consentimento OAuth" → tipo Externo → nome "Hollywood Studio AI" → seu email → salvar. Depois volte para Credenciais.
4
Stripe — cartão de crédito/débito
AGORA · 20 min
💳
Criar 3 Payment Links (R$99, R$199, R$349)
▸ Abrir Stripe → Criar Payment Link
Para cada plano, criar um link com:
• Produto: "Hollywood Studio AI — Básico" / "Premium" / "Avançado"
• Valor: R$99 / R$199 / R$349 (moeda: BRL — Real brasileiro)
• Após pagamento → redirecionar para:
https://hollywoodstudio.vercel.app?payment=success
Repita para os 3 planos. Copie cada link gerado (começa com https://buy.stripe.com/...)
Esses 3 links vão no admin do site em Pagamentos → Links Stripe por plano. Não precisa de código.
🔔
Configurar Webhook no Stripe
▸ Stripe → Webhooks → Add endpoint
URL do endpoint:
https://hollywoodstudio.vercel.app/api/stripe/webhook
Eventos para escutar: checkout.session.completed
Após criar, copie o Signing secret (whsec_...) e adicione como STRIPE_WEBHOOK_SECRET no Vercel.
Atenção: O webhook só funciona com o site em produção (não em localhost). Primeiro faça o deploy, depois configure o webhook com a URL real.
🔑
Pegar a Secret Key do Stripe
▸ Stripe → Developers → API Keys
Copie a Secret key (começa com sk_live_...) e adicione como STRIPE_SECRET_KEY no Vercel.
Use sk_live (produção), não sk_test. Clientes precisam pagar de verdade.
5
Pix — via MercadoPago (mais fácil para BR)
AGORA · 5 min
🏦
Opção mais simples: Link de pagamento MercadoPago
▸ MercadoPago → Cobrar → Criar link
1. Criar um link de pagamento genérico (sem valor fixo, para o usuário digitar)
2. Ou criar 3 links com os valores exatos (R$99, R$199, R$349)
3. Copiar o link gerado (formato: https://mpago.la/xxx)
No admin do site em Pagamentos → Pix, cole este link. Quando o usuário clicar em "Pagar com Pix", ele abre o MercadoPago que já oferece Pix + boleto + cartão.
Alternativa: Cole sua chave Pix diretamente (CPF, email, telefone ou chave aleatória). O botão vai copiar a chave para o usuário.
6
Admin do site — configurar tudo pelo painel
DEPOIS DO DEPLOY
🛡️
Acessar o admin
Entre no site com seu email Google (o mesmo cadastrado como ADMIN_EMAIL no Vercel). O menu Admin aparece automaticamente para seu usuário.
▸ hollywoodstudio.vercel.app → Login Google → Admin
⚙️
Sequência no admin
1. Admin → Integrações → Cole as chaves dos provedores (Atlas Cloud, BytePlus) → Salvar
2. Admin → Pagamentos → Cole os links Stripe + email PayPal + chave Pix → Salvar
3. Admin → Integrações → Campo GOOGLE_CLIENT_ID → Cole o Client ID → Salvar
4. Admin → Integrações → AUTH_ENDPOINT → Cole https://hollywoodstudio.vercel.app
5. Admin → Clique em ☁ Salvar no servidor → Digite a senha ADMIN_SECRET que definiu no Vercel
6. Pronto — toda nova sessão carrega essa config automaticamente para qualquer usuário
O botão "Salvar no servidor" só aparece depois de subir o admin-patch.js. Ele salva tudo no Vercel KV — sem FTP, sem editar arquivo.

7
Domínio hollywoodstudio.ai — conectar ao Vercel
DEPOIS DO TESTE
🌐
Onde o domínio está registrado?
Verifique no painel do seu registrador (GoDaddy, Registro.br, Namecheap, etc). É lá que você precisa apontar o DNS.
⚡
Adicionar domínio no Vercel
▸ Vercel → seu projeto → Settings → Domains → Add
Digite hollywoodstudio.ai e www.hollywoodstudio.ai. O Vercel mostra os registros DNS para adicionar.
No seu registrador de domínio: Adicione um registro do tipo A apontando para 76.76.21.21 e um CNAME www apontando para cname.vercel-dns.com. Propagação: até 24h (geralmente 5-10 min).
Depois de conectar o domínio: volte ao Google Cloud Console e adicione https://hollywoodstudio.ai como origem autorizada no OAuth. E atualize SITE_URL no Vercel para o domínio real.
